Why Session Dashboard?

AI coding agents work in the terminal. Humans need visual feedback. Session Dashboard bridges that gap.

🤖

Built for AI Agents

When AI agents like Claude Code work on your codebase, they make changes fast. Session Dashboard gives you a live view of what's happening without interrupting the agent's flow.

⚡

Real-time Updates

Uses Server-Sent Events (SSE) to push updates instantly. No manual refreshing needed. See commits, file changes, and PR status the moment they happen.

🔒

Completely Local

Runs entirely on your machine. Your code never leaves your computer. The dashboard talks directly to your local git repo and the GitHub API with your own token.

💬

Human-Agent Communication

Leave comments on specific files or lines. Agents can read them and respond. A native way to guide AI work without breaking their context window.

🔌

Extensible via Plugins

Need custom data? Write a plugin. The plugin API lets you inject any data you want into the dashboard—build notifications, test results, deployment status, anything.

🎯

Zero Config

Just run it in any git repo. It auto-detects your GitHub remote, finds your auth token from the gh CLI, and starts serving. No setup files, no environment variables required.

Features

Everything you need to monitor AI coding sessions

Branch & PR Overview

See your current branch, associated PR, and its status at a glance. Know immediately if your PR is in draft, ready for review, or merged.

  • Current branch and base branch display
  • PR number, title, and status
  • Draft state indicator
  • Direct links to GitHub PR page

Screenshot: Branch & PR Header

CI Status & Checks

Monitor all your CI checks in real-time. See which checks are passing, failing, or still running without leaving your dashboard.

  • Aggregated status (pass/fail/pending)
  • Individual check breakdown
  • Direct links to check details
  • Real-time status updates

Screenshot: CI Checks Panel

File Changes with Diffs

Browse all changed files with full diff views. See additions, deletions, and line-by-line changes for both committed and uncommitted work.

  • Committed changes (vs base branch)
  • Staged, unstaged, and untracked files
  • Full diff view with syntax highlighting
  • File-level addition/deletion stats

Screenshot: Files Tab with Diffs

Comments & Review Threads

View all PR review comments and local comment threads in one place. Reply to comments, resolve threads, and communicate with your team or AI agent.

  • GitHub PR review comments
  • Local comments (stored in repo)
  • Reply directly from dashboard
  • Resolve/unresolve threads

Screenshot: Comments Tab

Agent Tabs

AI agents can create custom HTML tabs that appear in the dashboard. Perfect for progress reports, visualizations, or any custom UI the agent wants to show you.

  • Agents drop HTML files in .sessiondashboard/tabs/
  • Auto-detected and rendered as tabs
  • Live updates when files change
  • Full HTML/CSS/JS support

Screenshot: Custom Agent Tab

PR Stack Visualization

Working with stacked PRs? See the entire stack at once. Understand which PRs depend on each other and navigate between them easily.

  • Current PR highlighted
  • Child PRs (based on this branch)
  • State indicators (open/merged/closed)
  • Quick navigation links

Screenshot: PR Stack View

Get Started in 30 Seconds

Session Dashboard works with any git repository. Just clone, install, and run.

# Clone the repo
git clone https://github.com/luigiinred/session-dashboard.git
cd session-dashboard

# Install dependencies
npm install && cd web && npm install && npm run build && cd ..

# Run in any git project
cd /path/to/your/project
npx agentdashboard -o

Requires Node.js 18+. For GitHub features, authenticate with gh auth login or set GITHUB_TOKEN.
